站点数据统计

通过站点统计，您可以整体把握站点的发展状况。

您还可以查看趋势统计，把握站点每日变化。

  • 开通空间数: $statistics[spacenum] (趋势)
  • 全部动态数: $statistics[feednum]
  • 全部公告数: $statistics[blognum] (趋势)
  • 全部相册数: $statistics[albumnum] (趋势)
  • 全部分享数: $statistics[sharenum] (趋势)
  • 全部话题数: $statistics[threadnum] (趋势)
  • 全部评论数: $statistics[commentnum]
  • 开启应用数: $statistics[myappnum]

程序数据库/版本

  • 操作系统: $os
  • 数据库版本: $statistics[mysql]
  • 上传许可: $fileupload
  • 数据库尺寸: $dbsize
  • 附件尺寸: $attachsize
  • 当前程序版本: UCenter Home $statistics[version] ( $statistics[release] )
  • UCenter Client 版本: {UC_CLIENT_VERSION} Release {UC_CLIENT_RELEASE}

欢迎光临管理平台

通过管理平台操作，你可以对发布的信息进行批量管理。


快捷管理菜单

  • 用户

    编辑用户的积分、用户组、管理空间信息、删除用户

  • 事件

    对"我在做什么"进行批量删除

  • 公告

    对公告进行批量删除、编辑

  • 相册

    对发布的相册进行批量删除

  • 图片

    对发布的图片进行批量删除

  • 评论

    对发布的评论进行批量删除

  • 话题

    对群组上面发布的话题进行批量删除、精华、置顶

  • 回帖

    对群组上面发布的回帖进行批量删除

  • 投票

    对投票进行批量删除

  • 标签

    对站点的标签进行删除、关闭、合并

  • 群组

    对站点的群组进行删除、关闭、合并

  • 活动

    对站点的活动进行删除、审核、推荐

  • 模版样式

    对站点共享的模版样式进行批量审核、删除
